# ✏️ Whiteboard

A clean, fast, canvas-based whiteboard application built with **Next.js** and **Tailwind CSS**, ready to deploy on **Vercel**.

## Features

- 🖊️ **Freehand drawing** with smooth strokes
- 🧹 **Eraser** tool
- 🎨 **Color picker** (9 presets + custom)
- 📏 **Stroke width** selector (4 sizes)
- ↩️ **Undo / Redo** (Ctrl+Z / Ctrl+Shift+Z)
- 🗑️ **Clear canvas** button
- 💾 **Download** whiteboard as PNG
- 📱 **Touch support** for tablets/phones
- ⌨️ Keyboard shortcuts

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/blippppp/whiteboard)

Or use the Vercel CLI:

```bash
npx vercel
```
